# BTBS - Bus Ticket Booking System

## Overview
This is a production-ready Bus Ticket Booking System built with:
- **Frontend**: Angular 19 + Tailwind CSS + PrimeNG
- **Backend**: Node.js + Express (API)

## Structure
- `/BTBS_WEB`: Angular application
- `/BTBS_API`: Express API

## Getting Started

### Web
1. `cd BTBS_WEB`
2. `npm install`
3. `npm start` (Runs on http://localhost:4200)

### API
1. `cd BTBS_API`
2. `npm install`
3. `npm run dev` (Runs on http://localhost:3000)

## Features
- Clean architecture
- Centralized icon management (Lucide)
- Scalable backend structure
